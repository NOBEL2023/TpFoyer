Extract consomme PSNEXT Script


This Python script extracts charge data from an Excel file based on specified conditions. It performs the following tasks:

Reads input Excel file and filters rows based on certain conditions.
Saves the filtered data to a new Excel file.
Extracts charge data from the intermediate Excel file.
Processes the data, including handling zero charges and dividing charges based on weekdays and months.
Outputs the final results to a CSV file.
